# Steps

## Preparation

- siapkan desain aplikasi
  
- install java
- install nodejs

- install react-native
- buat new project reat-native 
- run emulator from android studio

- install dependencies
    - axios
    - firebase
    - image-picker
    - redux
    - svg
    - document-picker
    - react-native-pdf
